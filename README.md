# ProjetoAgrinho2026
Este site foi desenvolvido como parte do Projeto Agrinho, com o objetivo de apresentar a importancia e as caracteristicas do agro no suldoeste do Paraná, destacando a agricultura, a sustentabelidade, a preservação ambiental e as tecnologias ultilisadas no campo de forma criativa e interativa.

O Projeto tambem serve como prática de progamação web ultilizando linguagem HTML, CSS, e JavaScript 

# FUNCIONALIDADES:
- Layout moderno e responsivo;
- Galeria de imagens do agro ;
- Cards informatimos;
- Navegação suave;
- Menu de acessibilidade;
- Efeitos visuais;
- Quiz interativo.
